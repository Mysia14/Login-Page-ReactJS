## :mag_right: How to install and use it.

Commands:

```bash
# Clone this repository
$ git clone https://github.com/SilvaneiMartins/react-login

# Go into the repository
$ cd react-login

# Install dependencies
$ npm install

# Run the app
$ npm start
```
